Process

First of all before I start to develop the website I create the mockup first. the tools that i use for creating the mockup of the web is Figma. For the Fonts I decided to use Roboto to give the professional type of fonts. For the color Pallete the reason I choose this color is to give my website kind of like dark mode view in order to not make my viewer dizzy if i choose more brighter colours.
